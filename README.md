# Digital Asset Protection System (DAPS)

## 📌 Overview
Digital Asset Protection System (DAPS) is an AI-based prototype that detects unauthorized usage of digital content such as images, videos, and documents.

---

## 🚀 Features
- Upload digital files
- AI-based scanning (simulation)
- Detection of unauthorized usage
- Real-time analysis animation
- Result display (Safe / Unauthorized)
- Downloadable report

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- AI Concept: Hash-based detection (simulation)
